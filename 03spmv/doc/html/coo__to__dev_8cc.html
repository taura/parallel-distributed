static int coo_to_dev ( sparse_t &  A)
static

make a deivce copy of a sparse matrix in coo format.

Parameters
(A)the reference to a matrix whose elems_dev has not been set (i.e., = NULL)
Returns
1 if succeed. 0 if failed.

this function allocates memory on the device and transfers A's non-zero elements in the allocated memory. it also should set A's elems_dev to the address of the allocated memory, so that if you pass A as an argument of a kernel launch, the device code can obtain all necessary information of A from the parameter.
