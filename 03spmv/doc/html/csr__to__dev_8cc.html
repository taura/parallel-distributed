static int csr_to_dev ( sparse_t &  A)
static

make a deivce copy of a sparse matrix in csr format.

Parameters
(A)the reference to a matrix whose elems_dev has not been set (i.e., = NULL)
Returns
1 if succeed. 0 if failed.

this function allocates memory blocks on the device and transfers A's row_start array and non-zero elements in the allocated blocks. it also should set A's elems_dev and row_start_dev to the addresses of the allocated blocks, so that if you pass A as an argument of a kernel launch, the device code can obtain all necessary information of A from the parameter.
